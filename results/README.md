## Results

### Blood Cell Detection

The YOLO-based detection pipeline successfully identified and counted:

- Red Blood Cells (RBCs)
- White Blood Cells (WBCs)
- Platelets

from microscopic blood smear images.

### Disease Diagnosis

The system analyzes extracted blood cell counts and predicts potential hematological conditions including:

- Anemia
- Leukopenia
- Blood Cancer
- Normal Blood Profile

### Performance

- Achieved **96.4% platelet detection accuracy**
- Automated blood cell counting and disease screening workflow
- Reduced manual analysis effort through end-to-end automation
