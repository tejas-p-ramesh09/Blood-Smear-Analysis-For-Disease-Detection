# Blood Smear Analysis for Hematological Disease Detection

## Overview

An AI-powered healthcare system that automatically detects and counts blood cells from microscopic blood smear images.

The system performs:

- RBC detection
- WBC detection
- Platelet detection
- Disease diagnosis
- Treatment recommendation
- PDF report generation

## Technologies Used

- Python
- YOLO
- Darkflow
- OpenCV
- Scikit-Learn
- Tkinter
- NumPy
- Pandas

## System Workflow

Blood Smear Image
→ Blood Cell Detection
→ RBC/WBC/Platelet Counting
→ Disease Diagnosis
→ Treatment Recommendation
→ PDF Report Generation

## Results

- Automated blood cell detection
- Automated disease diagnosis
- 96.4% platelet detection accuracy

src/ - application source code
